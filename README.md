# 📈 FinSight AI — Financial Reasoning Agent

An AI-powered financial analysis agent that synthesizes real-time stock data, news, and sentiment analysis into professional investment summaries.

## 🚀 Features

- **Candlestick & Trend Charts** — 3-month OHLCV candlestick, 1-year price trend, volume analysis
- **Key Fundamentals** — Market cap, P/E ratio, EPS, 52W range, beta, dividend yield, revenue
- **AI Sentiment Analysis** — News sentiment scored on a -10 to +10 gauge using Gemini 1.5 Pro
- **AI Investment Summary** — 3-paragraph professional investment summary generated by Gemini
- **Live News Feed** — Latest headlines fetched via Tavily or DuckDuckGo
- **Price History Table** — Last 25 trading days with OHLCV data

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Orchestration** | LangGraph |
| **LLM** | Google Gemini 1.5 Pro |
| **Stock Data** | yfinance (Yahoo Finance) |
| **News Search** | Tavily / DuckDuckGo |
| **Charts** | Plotly |
| **UI** | Streamlit |

## ⚙️ Setup

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/financial-reasoning-agent.git
cd financial-reasoning-agent
```

### 2. Create a virtual environment
```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure API keys
```bash
cp .env.example .env
```
Edit `.env` and add your keys:
```
GOOGLE_API_KEY=AIza...your_key_here
TAVILY_API_KEY=tvly-...optional
```

- 🔑 Get Google API key: [aistudio.google.com](https://aistudio.google.com/app/apikey)
- 🔍 Tavily (optional, DuckDuckGo used as fallback): [app.tavily.com](https://app.tavily.com)

### 5. Run the app
```bash
streamlit run main.py
```
Or double-click **`run.bat`** on Windows.

## 🖥️ Usage

1. Open [http://localhost:8501](http://localhost:8501) in your browser
2. Enter any stock ticker (e.g. `AAPL`, `MSFT`, `TSLA`, `GOOGL`)
3. Click **Analyze →**
4. View charts, metrics, sentiment, AI summary, and news
